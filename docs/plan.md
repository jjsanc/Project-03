# Project Plan
**Project Title:** Personal Collection 

---

# Project Overview
This project involves the design of a website that showcases personal collectio of:

- Pokemon cards
- Cologne (designer and niche fragrances)
- Collectible pins

---

## Project Objectives

- Develop a fuctional and responsive website
- Organize collection items into clear categories
- Publish the project using Github Pages

---

## Success Criteria

The project will be considered sucessful if:

- The website is fully functional
- All three collection categories are clearly displayed
- The project is documented with a professional ReaDME
- The website is successfully deployed online

---

## Constraints

- Limited development
- Limited technical resources

---

## Risk Analysis (TAME Framework)

## Technical Risks

| Risk | Impact | Probability | Score | Mitigation Strategy |

| Limited coding  knowledge | 4 | 5 | 20 | Keep design simple |
| Deplyment errors | 2 | 1 | 2 | Test before publishing |
